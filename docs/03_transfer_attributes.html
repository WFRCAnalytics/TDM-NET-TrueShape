True Shape Network

Transfer Model Attributes to True-Shape Links

  • Show All Code
  • Hide All Code
  • View Source

Pseudonode dissolution → piece graph traversal → true-shape geometry assembly

Model links are dissolved across pseudonode chains, then each dissolved link’s geometry is assembled by traversing the physical piece graph from A’s snap position to B’s snap position. Model attributes are transferred to the resulting true-shape LineStrings.

5 Part C — Assemble True-Shape Chains

For each dissolved link A→B, perform a constrained BFS through the piece graph from A’s snap coordinate to B’s snap coordinate. The BFS cannot traverse through real_junction nodes other than B — this ensures the path stays within the physical bounds of a single dissolved link.

Show the code 
print(f"Assembling {len(gdf_dissolved):,} dissolved links …")

trueshape_geometries: list = []
trueshape_methods: list[str] = []
n_unresolved = 0
n_no_path = 0

for i, (_, row) in enumerate(gdf_dissolved.iterrows()):
    if i > 0 and i % 2000 == 0:
        print(f"  {i:,} / {len(gdf_dissolved):,} …")

    a_n, b_n = int(row["A"]), int(row["B"])
    a_coord = node_coord_lookup.get(a_n)
    b_coord = node_coord_lookup.get(b_n)

    if a_coord is None or b_coord is None:
        n_unresolved += 1
        a_pt = node_geom_lookup.get(a_n)
        b_pt = node_geom_lookup.get(b_n)
        fallback = LineString([a_pt, b_pt]) if (a_pt and b_pt) else None
        trueshape_geometries.append(fallback)
        trueshape_methods.append("unresolved_no_snap")
        continue

    pieces = assemble_chain(piece_graph, a_coord, b_coord)

    if pieces is None:
        n_no_path += 1
        a_pt = node_geom_lookup.get(a_n)
        b_pt = node_geom_lookup.get(b_n)
        fallback = LineString([a_pt, b_pt]) if (a_pt and b_pt) else None
        trueshape_geometries.append(fallback)
        trueshape_methods.append("unresolved_no_path")
    else:
        merged = linemerge(pieces) if len(pieces) > 1 else (pieces[0] if pieces else None)
        trueshape_geometries.append(merged)
        trueshape_methods.append("dissolved")

n_dissolved = trueshape_methods.count("dissolved")
print(f"\nDissolved (true-shape):  {n_dissolved:,} ({n_dissolved/len(gdf_dissolved):.1%})")
print(f"Unresolved (no snap):    {n_unresolved:,}")
print(f"Unresolved (no path):    {n_no_path:,}")
